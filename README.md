# React + TypeScript + Vite + Tailwind CSS Project

This is a React project built with TypeScript, Vite, and Tailwind CSS.

## Features

- React 18
- TypeScript
- Vite for fast development
- Tailwind CSS for styling
- Dark mode support
- Custom animations

## Getting Started

### Prerequisites

- Node.js (v18 or newer recommended)
- npm or yarn

### Installation

1. Clone the repository:

```bash
git clone [repository-url]
cd [project-name]
```

2. Install dependencies:

```bash
npm install
```

### Development

Run the development server:

```bash
npm run dev
```

The app will be available at http://localhost:5173/

### Building for Production

Build the app for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Project Structure

- `src/` - Source code
  - `assets/` - Static assets
  - `components/` - React components
  - `App.tsx` - Main app component
  - `main.tsx` - Application entry point

## Customization

- Edit `tailwind.config.js` to customize your Tailwind CSS setup
- Modify `src/index.css` to add custom styles beyond Tailwind
